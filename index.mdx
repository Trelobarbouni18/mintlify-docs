## What you can do

Prefetch API lets you enrich any URL with structured data — all powered by headless Chrome for accuracy on JavaScript-heavy sites.

## All endpoints at a glance

| Endpoint | Credit cost | What it returns |
|---|---|---|
| [`GET /enrich`](/api-reference/endpoint/enrich) | 3 | Brand + company + classification combined |
| [`GET /brand`](/api-reference/endpoint/brand) | 2 | Colors, logos, fonts, social links |
| [`GET /company`](/api-reference/endpoint/company) | 2 | Company name, emails, addresses |
| [`GET /classify`](/api-reference/endpoint/classify) | 1 | IAB category and description |
| [`GET /screenshot`](/api-reference/endpoint/screenshot) | 1 | Webpage screenshot URL |
| [`GET /health`](/api-reference/endpoint/health) | — | Liveness check |
| [`GET /ready`](/api-reference/endpoint/ready) | — | Readiness check (Redis connectivity) |
